# 📘 Version 1 – Basic Java (Procedural Programming)

## Objective

Build a console-based Library Management System using only core Java fundamentals.

---

## Java Concepts Covered

- Variables
- Arrays
- Scanner
- Methods
- Loops
- Switch Case
- Conditional Statements

---

## Features

- Add Book
- Display Books
- Search Book
- View Book Details
- Delete Book

---

## Project Structure

Main.java

---

## Storage

```text
bookId[]
bookTitle[]
bookAuthor[]
```

Data is stored using parallel arrays.

---

## Learning Outcome

This version focuses on understanding procedural programming and solving problems using arrays and methods without Object-Oriented Programming.

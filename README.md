# 📚 Library Management System

A Java project built to learn Java programming by developing a Library Management System from scratch.

Instead of building everything at once, this project is developed in multiple versions. Each version introduces new Java concepts and improves the application design.

The goal is to understand not only *how* to write Java code but also *how software evolves* as new concepts are learned.

## ✨ Features

- Add Book
- Display Books
- Search Book
- View Book Details
- Delete Book
- Count Total Books

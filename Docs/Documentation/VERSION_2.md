# 📗 Version 2 – Object-Oriented Programming

## Objective

Refactor Version 1 into an Object-Oriented application without changing the functionality.

---

## Java Concepts Covered

- Classes
- Objects
- Constructors
- Encapsulation
- Getters & Setters
- Object Arrays
- Method Reusability
- toString()

---

## Project Structure

```text
Book.java

Library.java

Main.java
```

---

## Features

- Add Book
- Display Books
- Search Book
- View Book
- Delete Book
- Count Books

---

## Class Responsibility

### Book

Represents a single book.

Contains

- Book ID
- Title
- Author
- Availability

---

### Library

Responsible for managing books.

Contains

- Add Book
- Display Books
- Search Book
- View Book
- Delete Book

---

### Main

Acts as the entry point of the application.

Responsible for

- Displaying Menu
- Taking User Input
- Calling Library methods

---

## Learning Outcome

This version introduces Object-Oriented Programming by separating data and business logic into different classes.
